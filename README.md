藏文键盘打字练习程序 | Tibetan Keyboard Typing Tutor

📄 ངོ་སྤྲོད། | 简介 | Introduction

བོད་ཡིག:
འདི་ནི་བོད་ཡིག་གི་མཐེབ་གཞོང་འབྲི་རྩལ་སྦྱོང་བརྡར་བྱེད་པའི་དྲ་ཐོག་མཉེན་ཆས་ཤིག་ཡིན།
སློབ་ཚན་གྱི་སྒྲིག་གཞི་ནི་སླ་མོ་ནས་དཀའ་མོའི་བར་རིམ་པ་གསུམ་དུ་བགོས་ཡོད་པས་བོད་ཡིག་འབྲི་བཞིན་པའི་ལས་དང་པོ་བ་ནས་བྱང་ཆ་ལྡན་པ་ཚང་མར་འཚམ་པོ་ཡོད།

中文: 这是一个基于 Web 的专业藏文打字练习程序。它设计了从易到难的三个训练级别，旨在帮助用户系统地掌握藏文键盘布局，提高打字速度和准确率。

English: This is a professional web-based Tibetan typing tutor. Designed with
three proficiency levels, it helps users systematically master the Tibetan
keyboard layout, improving both speed and accuracy through interactive practice.

🌟 ཁྱད་ཆོས་གཙོ་བོ། | 主要功能 | Key Features

1. སློབ་ཚན་རིམ་པ་གསུམ། | 三大练习阶段 | Three Practice Levels

  - Level 1: ལས་དང་པོ་བ། (初学者/Beginner)
      - གསལ་བྱེད་རྐྱང་པ་དང་དབྱངས་བཞི། མཐེབ་གནས་ངོས་འཛིན།
      - 基础字母与元音，熟悉键盘位置。
      - Basic characters and vowels; mastering key positions.
  - Level 2: རྨང་གཞིའི་རིམ་པ། (基础进阶/Intermediate)
      - མགོ་འདོགས་བརྩེགས་མ་དང་རྒྱུན་བཀོལ་ཐ་སྙད།
      - 叠字、前后加字及常用词汇练习。
      - Stacked characters, prefixes, suffixes, and common vocabulary.
  - Level 3: བྱང་ཆ་ལྡན་པ། (高级挑战/Advanced)
      - བོད་ཀྱི་ལེགས་བཤད་དང་ལྷུག་རྩོམ་རིང་མོ།
      - 藏文经典格言与散文长文本练习。
      - Famous Tibetan literature, proverbs, and long prose practice.

2. ལག་རྩལ་གྱི་ཁྱད་ཆོས། | 技术亮点 | Technical Highlights

  - Interactive Guide (མཛུབ་མོ་སྟོན་པ།):
    མཐེབ་གཞོང་སྟེང་མཛུབ་མོ་གང་གིས་མཐེབ་གནས་གང་གནོན་དགོས་པའི་འགུལ་རིས་ལམ་སྟོན།
    (动态指法引导，实时显示按键指法。) Real-time animated finger guide for correct typing
    posture.
  - Real-time Stats (གནས་སྡུད་བརྩི་བ།): དུས་ཚོད། ཡིག་གྲངས།
    འགྲིག་ཚད་བཅས་ཐད་ཀར་བརྩི་ཐུབ།
    (实时统计时间、字数及准确率。) Track time, keystrokes, and accuracy instantly.
  - Themes (མཐོང་སྣང་བརྗེ་བ།): ཉིན་མོ་དང་མཚན་མོའི་ (Dark Mode)
    མཐོང་སྣང་བརྗེ་ཆོག་པ། (支持昼夜模式切换。) Toggle
    between Day and Night (Dark Mode) themes.
  - Feedback (བསམ་འཆར་ཡར་ཞུ།):
    བསམ་འཆར་ཡར་ཞུའི་སྐར་ཁུང་བརྒྱུད་ནས་བཟོ་བཅོས་རེ་བ་ཞུ་ཆོག
    (内置反馈系统。) Integrated feedback system via Formspree.
